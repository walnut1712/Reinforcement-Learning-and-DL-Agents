# 🤖 Reinforcement Learning and Deep Learning Agents

## 📌 Project Overview

Reinforcement Learning (RL) is a key technique used in many real-world autonomous systems, including robotics, self-driving cars, drone navigation, warehouse automation, adaptive game AI, traffic control, and simulation-based training.

This project provides hands-on experience designing and training intelligent agents that learn through interaction with their environment. You will build **two interactive environments in Python** and train agents using both **classical reinforcement learning** and **deep reinforcement learning** techniques.

By completing this project, you will gain experience building intelligent systems from the ground up, including designing environments, defining rewards, training agents, and analysing learning behaviour.

---

# 🧠 Project Structure

The assignment is divided into two main parts:

* **Part I – Classical Reinforcement Learning (Gridworld)**
* **Part II – Deep Reinforcement Learning (Pygame Arena)**

Each part focuses on different techniques and levels of complexity in reinforcement learning.

---

# Part I: Gridworld – Value-Based Reinforcement Learning

## 🎮 Environment

The first environment is a **visual Gridworld**, where an agent moves across a grid to reach goal states while avoiding penalties.

Each cell represents a **state**, and the agent can perform actions such as moving **up, down, left, or right**.

The agent learns by receiving **rewards or penalties** depending on its actions and final outcomes.

---



# Part II: Pygame Arena – Deep Reinforcement Learning

## 🎮 Environment

The second part expands the project into a **real-time interactive Pygame environment**.

In this arena, an agent must learn behaviors such as:

* Movement and navigation
* Avoiding obstacles or threats
* Surviving for longer durations
* Progressing toward objectives

Unlike the Gridworld environment, this simulation runs **continuously in real time**, requiring the agent to learn dynamic behavior.

---

## 🧠 Deep Reinforcement Learning

This part introduces **Deep Reinforcement Learning (DRL)** using the **Stable Baselines3** framework.

Deep RL combines reinforcement learning with **neural networks**, allowing agents to learn directly from complex environments.

The neural network learns to approximate policies or value functions based on observations from the environment.

This type of approach is widely used in:

* Game AI development
* Robotics control systems
* Autonomous vehicles
* Simulation and training systems



# 📊 Training and Evaluation

During training, the agent interacts with the environment and gradually improves its policy.

The system records **training logs and performance metrics**, allowing analysis of:

* Learning progress over time
* Reward improvements
* Policy behaviour
* Stability of training

These logs help evaluate whether the agent is successfully learning the desired behaviours.
